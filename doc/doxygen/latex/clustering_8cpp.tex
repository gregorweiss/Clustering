\subsection{Detailed Description}
The main function of 'clustering' is essentially being a wrapper around the different sub-\/modules\-:
\begin{DoxyItemize}
\item {\bfseries density}\-: for density-\/based clustering on the given geometric space
\item {\bfseries network}\-: for the network/microstate generation from density-\/based clustering results
\item {\bfseries mpp}\-: for Most Probable Path clustering of microstates
\item {\bfseries coring}\-: for boundary corrections of clustered state trajectories
\item {\bfseries filter}\-: for fast filtering of coordinates, order parameters, etc. based on\par
 a given state trajectory (i.\-e. clustering result) 
\end{DoxyItemize}
